## Getting Started
### üíª  Set up autodeploys
- Install our Github Bot to setup autodeploying your docs. You should have received an email with a link. Didn't get an email? <a href="mailto:kate@mintlify.com">Contact us.</a>
- Download our <a href="https://mintlify.com/docs/development">Mintlify CLI</a> to preview changes locally.

### üòé Make it yours
1. Change `name` in the `mint.json` file to your company or product's name.
2. Update `logo` folder with the light and dark version of your logo (can be svg or png).
3. Populate the `colors` object with colors that match your brand
4. Change the `favicon.png` with an image of the favicon
5. Update `footerSocials` with the right `[platform]: [url]` key-value pair

<Tip>

</Tip>
Add anything from images and new pages on your docs to creating your own MDX components. Check out the components we have built out of the box <a href="https://mintlify.com/docs/writing-content/overview">here</a>.<br />

Have existing docs and want support migrating? <a href="mailto:kate@mintlify.com">Contact us.</a>
